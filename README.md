#  Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript.

## Features

- Snake movement using arrow keys
- Random food generation
- Score tracking
- High score saved using Local Storage
- Collision detection
- Background music and sound effects
- Game Over detection

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Poojasaurus/snake-game.git
```

2. Open `index.html` in your browser.
